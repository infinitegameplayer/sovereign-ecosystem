# Getting Started

This folder walks you through the first build of the Sovereign Ecosystem.

The sessions are meant to be completed across multiple sittings.
Do not try to force the whole system into one afternoon.

This is not a curriculum to finish.
It is a paced build sequence for getting the system operational.
Each session should leave you more able to actually use the ecosystem, not just understand it conceptually.

## Orientation

Start by locating where you actually are.

Useful questions early:
- Have you used tools like this before?
- Have you used AI before?
- How much have you used AI?
- What kinds of workflows or tools are already familiar to you?

That context matters.
Some people will move quickly.
Some people will need more analogy, more examples, and more time for the tools to become intuitive.

The point of these sessions is not to memorize the whole ecosystem.
The point is to build enough structure that the system starts helping you carry itself in real life.

This build should not only optimize how you have always done things.
It should help reveal how your more authentic, alive, curious way of working wants to function.

Build the foundation first.
Then let the system meet real life.
