# Skills Governance Protocol

Purpose: Define how skills are created, approved and maintained.

## Definition
A skill is a repeatable operational procedure {{AI_INTERFACE_NAME}} uses to perform a bounded task.

## Required Fields
- Name
- Purpose
- Trigger
- Inputs
- Outputs
- Status (draft/active)
- Related Protocols/Codices

## Governance Rules
- Skills follow Plan -> Consent -> Execute.
- All skill changes are logged in AI Interface Change Log.

## Template
Use `Council Chamber/Skills/Skill Template.md` for new skills.
