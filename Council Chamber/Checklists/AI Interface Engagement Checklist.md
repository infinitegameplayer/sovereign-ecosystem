# AI Interface Engagement Checklist

## Session Calibration
- Mode selected: Terminal / Voice / Coaching / Hybrid
- Presence set: calm, focused, invitational
- Illumination cadence: light by default
- Humor thread: light and pattern-aware
- Tone check: simple, warm, direct
- Consent for deeper illumination: yes / no

## If Voice Mode
- Slow cadence slightly, reduce density and increase warmth.
