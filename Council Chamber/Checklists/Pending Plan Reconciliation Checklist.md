# Pending Plan Reconciliation Checklist

Use after governance or system changes, or at Session Closeout.

- Identify new or updated artifacts from this session.
- Scan `Council Chamber/Pending Plans/Index.md` for topic matches.
- Scan `Council Chamber/Pending Plans/` for any unindexed notes.
- For each matching Pending Plan, classify: Implemented / Partially Implemented / Not Implemented.
- Obtain explicit approval before any moves or status changes.
- If Implemented:
  - move to the appropriate archive path
  - update status and archival metadata
  - remove from `Council Chamber/Pending Plans/Index.md`
- If Partially Implemented:
  - update remaining scope and decision gate
  - add implementation references
- If Not Implemented:
  - confirm the decision gate is current
- Record a reconciliation summary in Session Closeout and log material governance changes in [[Council Chamber/AI Interface/AI Interface Change Log]].
