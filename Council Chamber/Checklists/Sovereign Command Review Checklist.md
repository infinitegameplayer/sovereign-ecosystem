# Sovereign Command Review Checklist

Purpose: Keep `Sovereign Command.md` high-signal, readable and aligned to the protocol.

## Checklist
- `Most Alive Next Move (Primary)` reflects the current highest-energy move.
- The note remains readable in under 2 minutes.
- Optional sections are omitted when they have no live candidate.
- Admin items appear only when they materially support the primary move.
- Approvals are surfaced only when they are actually waiting on the Sovereign.
- The note does not duplicate deep project plans, protocol guidance or North Star detail.
- Every promoted item has a clear source note behind it.
- Proposed updates remain approval-gated.
