# Inbox Intake Analysis Template

Purpose: Provide tiered analysis that scales with input size and impact.

## Tier A - Minimal
- Summary (1-2 lines)
- Type suggestion
- Status suggestion

## Tier B - Standard
- Summary
- Potential use cases
- Suggested object + domain
- Proposed status
- Route recommendation

## Tier C - Deep
- Summary
- Key insights
- Potential use cases
- Proposed insertions/updates
- Risks & assumptions
- Council Chamber/Pending Plans/Quests suggested
- Approval gate

## Preservation Rule
- Keep analysis at the top of the note while in Inbox.
- When routed, move to an "Intake Analysis (Archive)" section or wrap in HTML comment.
