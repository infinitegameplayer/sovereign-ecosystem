# Inbox

Purpose: landing zone for all incoming material before classification and routing.

All new captures land here first.
Nothing is analyzed here — only held until the AI interface processes it.

## How This Works

1. Drop any new material here as an InboxItem.
2. During your next session, the AI processes each item using the Capture Classify Route Protocol.
3. Items get classified, routed to the right container, and cleared from the Inbox.

Use: [[Council Chamber/Templates/Object Templates/InboxItem]]

## Protocol

See: [[Council Chamber/Protocols/Inbox/Capture Classify Route Protocol]]

## Rule

Inbox is a transit zone, not a storage zone.
If an item has been here for more than a week without classification, it wants attention.
